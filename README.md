## 👨‍💻 About

This is my personal developer portfolio built with vanilla HTML, CSS and JavaScript. It showcases my skills, projects, education and certifications — with a clean dark UI, smooth animations and a fully responsive layout.

---

## ✨ Features

- ⚡ Fully responsive — works on mobile, tablet and desktop
- 🎨 Dark themed UI with smooth scroll animations
- 🗂️ Filterable project cards (Backend / AI / Full Stack)
- 📋 One-click email copy
- 📬 Contact form with validation
- 🔧 Easy to customize via a single `data.js` file

## 📁 Project Structure

```
Personal-Portfolio-Website/
│
├── index.html          # Main HTML structure
├── style.css           # Global styles
├── components.css      # Component-level styles
├── variable.css        # CSS custom properties / theme
├── script.js           # Rendering logic & interactions
├── data.js             # ⭐ All content lives here (edit this!)
│
└── assets/
    └── profile-photo.jpg   # Your profile photo
```

---

## ⚙️ How to Customize

All content is managed from a **single file** — `data.js`

```js
window.PORTFOLIO = {
  profile: {
    name: "Your Name",
    role: "Your Role",
    photo: "./assets/profile-photo.jpg",
    links: {
      github: "https://github.com/your-username",
      linkedin: "https://linkedin.com/in/your-profile",
      resume: "https://drive.google.com/your-resume",
    }
  },
  skills: [ ... ],
  projects: [ ... ],
  education: [ ... ],
  certifications: [ ... ],
}
```
